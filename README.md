# 🧪 Flaky Test Detection Using Machine Learning

## 📌 Overview
Flaky tests are automated test cases that produce inconsistent results without any code changes.  
They increase CI/CD noise, waste execution time, and reduce confidence in automation.

This project uses **Machine Learning and Time-Series Analysis** to **automatically detect flaky test cases** based on historical execution data and integrates the results into a CI/CD pipeline.

---

## 🎯 Objectives
- Identify unstable (flaky) test cases automatically  
- Reduce false failures in CI pipelines  
- Improve reliability and trust in automation testing  
- Demonstrate the use of **Data Science in Automation Testing**
